# Organizador-de-Declaracao-de-Imposto-de-Renda

Este projeto foi desenvolvido como parte de um desafio do Bootcamp Santander - Excel com Inteligência Artificial.  
O objetivo é facilitar a organização das informações necessárias para a declaração do Imposto de Renda, reunindo dados pessoais, 
informes bancários e registros de receitas em uma única planilha.

## Funcionalidades
- Cadastro do titular: nome, CPF, data de nascimento, endereço, contatos e informações adicionais.
- Informes de rendimentos bancários: registro de bancos, valores atuais e anexos.
- Notas bancárias / holerites: entradas mensais de receita com data, categoria e valor.

## Como usar
1. Abra o arquivo `_Organizador de Declaração de Imposto de Renda.xlsx` no Excel.
2. Preencha os dados do titular na primeira aba.
3. Insira os informes de rendimentos bancários na aba correspondente.
4. Registre as entradas mensais de receita na aba de notas.
5. Utilize a planilha como base para organizar e preparar sua declaração anual.
